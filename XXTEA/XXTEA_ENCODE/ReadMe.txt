Задача 1 : реализация симметричного шифра (шифрование)
Алгоритм : XXTEA
Выполнил : Кошелев Георгий, КБ-501
Платформа: семейство Windows
Исх.файл : main.asm
Ассемблер: Microsoft (R) Macro Assembler Version 11.00.50727.1
Линкер   : Microsoft (R) Incremental Linker Version 11.00.50727.1
Сборка	 : вызов build.cmd
Исп.файл : main.exe
Вх.парам.: в одной директории с main.exe должны находиться файлы "key" - ключ , и "input" - открытый текст
Результат: результат шифрования будет записан в файл "output"
Зависимости: библиотека Irvine32 для осуществления операций записи и чтения из файлов, которая в свою очередь зависит еще от нескольких библиотек.
	     все зависимости находятся в папке deps.
Примечание: для сборки используется ассемблер и линкер, которые идут в комплекте с Visual Studio 2012.
	    Если VS2012 не установлена, следует заменить пути до ml.exe и link.exe в build.cmd
